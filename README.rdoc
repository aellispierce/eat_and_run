#EAT AND RUN...IDEAS FOR LATER

-User can choose activity: walking, running, or biking
-User can enter personal height/weight information for better calorie calculation
-User can enter zip code, which would get translated into approx. lat/long, which would allow them to specify a certain radius for how far they were willing to travel.  A lat/long to address conversion or link to Google Maps would help them locate the trails.
-Connect with an API to provide food images
-Connect with a weather API to give current temperature/precipitation for the trail
-Connect with restaurant API to give restaurant recommendations
-Could include maps
